# LPIPS在 0~1 的范围内，0.07 属于非常低的水平，说明你的超分辨率结果在纹理、结构、细节上与真实高分辨率图像极为接近，人眼很难分辨差异。作为对比，常见 NeRF 超分方法在相同数据集上 LPIPS 通常在 0.09~0.12 之间，0.07 表明感知质量显著优于多数基线。

# NIQE 是无参考指标，评估图像的自然度（是否像真实照片）。数值越低表示图像越自然，一般真实图像的 NIQE 在 3~6 之间。NIQE大于7之后图像效果较差
